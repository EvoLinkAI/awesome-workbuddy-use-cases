## 这是什么？

WorkBuddy 不是聊天机器人。它是一个能操控浏览器、读写本地文件、执行代码、调用 API、完成复杂多步骤工作流的**数字员工**。

这个仓库是一个精心维护的 **230+ 真实自动化案例库**，按职业和任务类型组织。每个案例都是即用型场景模板：复制描述，交给 WorkBuddy，坐等它完成。

<summary><b>🏢 通用办公与管理类</b> — 6 种职业</summary>

### 1. 财务人员

- **多源对账：** 自动下载银行流水与本地 ERP 导出表，匹配每一笔款项并标出差异。
- **发票自动归档：** 批量识别文件夹内的 PDF 发票，按月份、税额提取到 Excel 并重命名文件。
- **税务政策速报：** 每天检索税务局官网，摘要与本公司相关的政策变动。

### 2. HR / 招聘者

- **简历自动化筛选：** 批量读取下载文件夹里的 50 份简历，根据岗位要求输出评分表。
- **入职包准备：** 给定姓名，自动生成 Offer、入职须知、合同模版并填充关键信息。
- **面试调研：** 面试前，自动搜索候选人的社交媒体、公开作品集或行业评价。

### 3. 法务 / 律师

- **合同合规审查：** 读取合同文件，对比公司标准模版，标出所有"责任豁免"和"违约条款"的差异。
- **法律检索：** 根据案件背景，自动去裁判文书网搜索相似判例并总结法官判决逻辑。
- **证照到期提醒：** 扫描公司证照文件夹，自动生成一份到期时间表并关联日历。

### 4. 行政专员

- **资产自动化盘点：** 自动比对采购导出表和部门提交的登记表，标出未登记或遗失的设备条目。
- **差旅核算与分类：** 提取一整个文件夹的差旅发票 PDF，自动分类为机票、酒店、餐饮，并汇总报表。
- **会议室调度协助：** 根据群聊记录或指令，自动查询日历空闲时段，发送会议邀请并预定会议室。

### 5. 采购专员

- **多源报价横评：** 读取多家不同格式的供应商报价单，提取核心参数和价格，生成标准化对比 Excel。
- **资质合规扫描：** 扫描供应商提交的资质文件，自动联网查询企业工商信息是否异常。
- **订单物流追踪：** 每天定时访问物流后台抓取采购物资的最新状态，自动更新本地供应链看板。

### 6. 高管助理

- **碎片邀约转日程：** 读取微信截图或邮件里的碎片化邀约，自动生成结构化的每日日程表。
- **邮件智能草拟：** 每天早上对收件箱进行分类，并为常规询问邮件自动生成得体的回复草稿。
- **跨时区会议计算：** 识别外宾所在时区，结合老板本地日历，自动推荐双方都在工作时间内的会议时段。

### 7. 新媒体运营

- **爆款选题库：** 定时抓取小红书、抖音热榜，提取本领域热门话题并生成 5 个改写方向。

### 8. 电商运营

- **差评监控预警：** 自动监测店铺后台，发现 3 星以下评价立即通知并生成回访脚本。
- **竞品价格追踪：** 每 4 小时查看竞品 SKU 价格变动，自动更新本公司的调价建议表。
- **详情页文案：** 输入产品卖点，自动生成不同尺寸、不同促销风格的详情页描述。

### 9. 销售代表

- **"开场白"定制：** 访问潜在客户的官网，根据其最新新闻生成个性化的拜访邮件。
- **销售机会预测：** 分析 CRM 中的历史互动频率，标记出本周最可能成交的客户。
- **会议录音转方案：** 结束后将录音交给 WorkBuddy，自动提取痛点并生成初版报价单。

### 10. 公关专员

- **舆情监控预警：** 定时抓取媒体池中带有品牌词的报道，情感分析判定为"负面"即截图预警。
- **通稿多版本转换：** 将内部通告改写为正式公关稿，并生成适合不同社交平台的文案。
- **媒体库增量更新：** 从网上的行业榜单中自动提取博主信息，更新到本地的媒体库 Excel 中。

### 11. SEO / SEM 优化师

- **长尾词挖掘清洗：** 自动抓取长尾词、下拉词，执行去重和清洗后保存为 CSV。
- **竞品创意盗摄：** 自动搜索特定关键词，截取竞品投流的广告创意和着陆页文案。
- **数据异常归因：** 对比后台流量报表，自动找出转化率突降的具体广告计划或渠道。

### 12. 客服主管

- **工单情感打标：** 读取每日客服聊天记录导出表，识别出"愤怒"或"有投诉倾向"的对话。
- **话术库动态沉淀：** 从最新解决的疑难工单中提取优秀回复，总结并添加到共享话术库文档。
- **退换货状态自动回复：** 识别包含物流单号的聊天记录，自动查询 ERP 系统状态并生成回复。

### 13. 市场调研员

- **财报核心数据扒取：** 自动下载同行业公司的最新 PDF 财报，提取营收、利润等数据并作图。
- **问卷脏数据清洗：** 批量处理问卷 Excel，自动修正错别字、剔除无效问卷。
- **前沿趋势翻译提炼：** 抓取外网最新外文长篇研报，全文翻译并提炼核心结论做成 PPT。

### 14. 程序员

- **PR 自动审查：** 检查本地代码库的新增提交，根据企业规范进行代码质量评估。
- **技术文档同步：** 根据代码变更，自动更新本地的 README 或 API 开发文档。
- **环境快速搭建：** 一句指令执行复杂的命令行组合，安装依赖并配置本地变量。

### 15. 产品经理

- **竞品分析报告：** 自动搜索特定竞品的最新版本更新日志，总结其战略意图。
- **PRD 自动补全：** 给定核心逻辑，自动生成详细的边界条件判定和异常处理流程。
- **反馈聚合：** 将收集到的多份用户调研表汇总，提取排名前 3 的功能需求点。

### 16. 数据分析师

- **报表自动化：** 每天定时打开本地数据库或 Excel，更新仪表盘图表并截图发给团队。
- **异常数据检测：** 自动扫描销售额，发现环比波动异常时，自动查找潜在原因。
- **SQL 编写执行：** 用自然语言提问，自动生成 SQL 语句并直接导出结果到桌面。

### 17. 文案策划

- **多语种改写：** 将本地产品方案一键转化为中英日三语的社交媒体贴文。
- **SEO 关键词填充：** 自动优化现有文章，在不改变本意的情况下合理嵌入高热度搜索词。
- **品牌声量周报：** 搜索过去 7 天品牌在全网的讨论词云和舆情导向。

### 18. UI / UX 设计师

- **素材库自动整理：** 批量将导出的切图文件，根据其视觉内容自动重命名并分拣到对应文件夹。
- **文案占位符替换：** 读取产品需求文档，生成真实文案，替换掉设计稿里的测试文字。
- **竞品视觉巡检：** 自动访问竞品网站，截取并保存特定模块（如登录页）截图到本地参考库。

### 19. 视频剪辑 / 编导

- **长视频自动"找梗"：** 处理长视频的音频转写文本，标记出笑声、高频词出现的时间戳。
- **多语言字幕流水线：** 提取初版中文字幕，自动翻译并严格导出带时间轴的 SRT 文件。
- **脚本空镜头匹配：** 根据本地文字分镜脚本，自动去免版权网站搜索对应空镜头素材。

### 20. 翻译人员

- **带排版批量机翻：** 将文件夹内不同格式的 Word/PPT 进行首轮机翻，且不破坏原有排版结构。
- **术语对齐与提取：** 比对原文与人工定稿的译文，自动提取专业名词对照，丰富本地术语库。
- **本地化语境润色：** 将直译的英文产品介绍，按母语者的表达习惯进行口语化和营销化改写。

### 21. 学术研究员 / 研究生

- **文献批量扫读：** 给定包含 PDF 论文的文件夹，自动提取 Abstract 和 Method，汇总成综述表。
- **图表数据逆向恢复：** 识别旧论文中的柱状图/折线图截图，将其转化为近似的 Excel 数据表格。
- **引文自动格式化：** 将粗糙的网址或书名整理成符合 APA 或 IEEE 标准的规范参考文献列表。

### 30. 心理咨询师 / 社工

## 如何使用这份清单

1. **找到你的职业** → 跳转到[按职业分类](#按职业分类)，复制场景描述直接交给 WorkBuddy。
2. **按任务类型筛选** → 使用[按任务类型分类](#按任务类型分类)，找到你想自动化的工作类型。
3. **构建自定义 Agent** → 使用 [Agent 能力图谱](#agent-能力图谱)识别你的 Agent 需要哪些工具。
4. **深度研究** → [白皮书表格](use-cases/whitepaper/README.zh-CN.md)包含 100 个案例的完整结构化详情。
5. **参与贡献** → 通过 [CONTRIBUTING.zh-CN.md](CONTRIBUTING.zh-CN.md) 提交你的新案例。
